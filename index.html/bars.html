Stacked Bar Chart – Packet Count by Protocol and Label

This stacked bar chart shows total packet counts for each protocol (DNS, FTP, HTTP, HTTPS, MQTT, SMTP, SSH), divided into Normal and Attack traffic. The height of each bar represents the overall packet volume for that protocol, while the colored segments reveal how much of that traffic is normal vs. attack. In our dataset, normal traffic dominates for every protocol, but a noticeable portion of attack packets is still present.

Average CPU Usage Over Time of Day

This line chart tracks the average CPU usage for each device type over the same time-of-day categories. It highlights when devices are most active computationally. For instance, some devices show peaks in the evening, suggesting heavier workloads or user activity during that period.
