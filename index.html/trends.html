Packet Count vs Time of Day (By Device Type)

The stacked area chart shows how total packet counts vary across the time-of-day categories (Afternoon, Evening, Morning, Night) for different device types such as Camera, Laptop, Phone, Sensor, and Smart_TV. The height of the stack at each time period reflects the overall traffic volume, while each colored band reveals which devices contribute most at different times.

Average CPU Usage Over Time of Day

This line chart tracks the average CPU usage for each device type over the same time-of-day categories. It highlights when devices are most active computationally. For instance, some devices show peaks in the evening, suggesting heavier workloads or user activity during that period.
